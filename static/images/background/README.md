## 圖片要求
- **尺寸**: 建議 1920x1080 或 1600x900
- **格式**: JPG 或 PNG
- **檔案大小**: 建議 < 2MB 以保持載入速度
- **風格**: 等距視角（Isometric view）的插畫或數位藝術風格

## 如何生成圖片

1. **Midjourney** - https://midjourney.com
2. **DALL-E 3** - https://openai.com/dall-e-3
3. **Stable Diffusion** - https://stablediffusionweb.com
4. **Leonardo.ai** - https://leonardo.ai

或者從免費圖庫網站下載類似圖片：
- Unsplash: https://unsplash.com
- Pexels: https://pexels.com
- Pixabay: https://pixabay.com
